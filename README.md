# portdraw
> [!NOTE]
> portdraw is an internal library for the [aecgd project](https://github.com/aecgd/aecgd). Currently, portdraw is not meant to be
> used on other projects, but that will change soon as the project evolves.


A graphics library that you can run on the most resource constrained devices efficiently, including the ATMega328P (which contains only 2048
bytes RAM), exploiting optimizations that significantly boost speed an reduce RAM usage compared to other libraries.

If you're submitting changes to the graphics library used in aecgd that are not tied to the aecgd project, please send it here
rather than the main repository.
